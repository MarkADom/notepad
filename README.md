# notepad

A simple Notepad application using JavaFX.

## Description
The notepad project is a desktop application just like the notepad or text editor we usually see on our desktop. 

## Project Description
 - Manu Bar
    - New File
    - Open File
    - Save
    - Save As...
    - Exit
  
 - Edit
    - Select All
    - Copy
    - Cut
    - Paste
    - Search
